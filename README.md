# Serveur-Multimedia
Jellyfin docs: https://jellyfin.org/docs/


1:
Lancer votre raspberry avec une distribution debian sans interface graphique (exemple: raspberry pi os lite)

2:
mettez a jour avec ceci:
sudo apt update && sudo apt upgrade -y

3:
tapez la commande pour utiliser le script prevue:
curl https://repo.jellyfin.org/install-debuntu.sh | sudo bash

4:
sur un autre ordinateur taper http://Adresse IP du Raspberry Pi:8096
(vous pouvez voir l'addresse ip de votre raspberry pi avec la commande: ip -a)

5:
bien jouer vous avez votre netflix local sur votre réseau

(si probléme regarder ce guide: https://raspberrytips.fr/jellyfin-raspberry-pi/)
